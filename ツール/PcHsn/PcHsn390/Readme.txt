●インストールについて
   PcHusen のインストールは、32bit OS と 64bit OS とで使用するファイルが違
います。（64bit OS は x64 のみサポートしています）
   Windows XP/Vista/7/8/8.1 32bit には PcHsn390.exe を実行してインストール
してください。
   Windows XP/Vista/7/8/8.1 (x64 Edition) 64bit には PcHsn390(x64).exe を
実行してインストールしてください。
